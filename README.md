# react-poc
React Proof Of Concepts

### Prerequisites
* Install [NodeJS](https://nodejs.org/en/download/) and :
```bash
npm install -g http-server
```

### Installation
Clone the repo: 
```bash
git clone https://github.com/aplahiru/react-poc.git
```
### Select Project
Clone the repo: 
```bash
cd <select-project>
```

Installing dependencies: 
```bash
npm install
```

### Start
Let's start up, run:
```bash
npm start
```

Finally go to:
```bash
http://localhost:3000
